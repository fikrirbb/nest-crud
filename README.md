## Test Project CRUD NestJS

## Installation

```bash
$ npm install
```

Update .env
```bash
DB_HOST=localhost
DB_USERNAME=root
DB_PASSWORD=
DB_NAME=test-nest
SERVER_PORT=3001
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

graphql
```bash
http://localhost:3001/graphql
```
